## API

### Props

| Name     | Description        | Type      | Default |
| -------- | ------------------ | --------- | ------- |
| copy     | text can be copied | `boolean` | `false` |
| ellipsis | text ellipsis      | `boolean` | `false` |
| width    | text width         | `string`  | -       |

### Slots

| Name    | Description       |
| ------- | ----------------- |
| default | customize content |
