## API

### Props

| Name     | Description            | Type                  | Default    |
| -------- | ---------------------- | --------------------- | ---------- |
| trigger  | how to trigger tooltip | `'hover' \| 'click'`  | `'hover'`  |
| position | tooltip position       | `'bottom' \| 'right'` | `'bottom'` |
| text     | tooltip text           | `string`              | -          |

### Slots

| Name    | Description       |
| ------- | ----------------- |
| default | customize content |
| tooltip | customize tooltip |
