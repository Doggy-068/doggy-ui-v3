## API

### Props

| Name                  | Description     | Type                                     | Default   |
| --------------------- | --------------- | ---------------------------------------- | --------- |
| model-value / v-model | show drawer     | `boolean`                                | -         |
| position              | drawer position | `'top' \| 'right' \| 'bottom' \| 'left'` | `'right'` |
| title                 | drawer title    | `string`                                 | -         |

### Slots

| Name    | Description       |
| ------- | ----------------- |
| default | customize content |
| title   | customize title   |
