## API

### Props

| Name      | Description                | Type                   | Default     |
| --------- | -------------------------- | ---------------------- | ----------- |
| size      | tag size                   | `'default' \| 'small'` | `'default'` |
| color     | customize background color | `string`               | -           |
| closeable | is tag closeable           | `boolean`              | `false`     |

### Events

| Name  | Description | Params |
| ----- | ----------- | ------ |
| close | closed tag  | -      |

### Slots

| Name    | Description       |
| ------- | ----------------- |
| default | customize content |
