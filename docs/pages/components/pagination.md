## API

### Props

| Name                  | Description                  | Type     | Default |
| --------------------- | ---------------------------- | -------- | ------- |
| total                 | the total of data            | `number` | -       |
| size                  | the number of data in a page | `number` | `10`    |
| index / v-model:index | current page number          | `number` | -       |
