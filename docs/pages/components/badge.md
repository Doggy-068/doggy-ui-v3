## API

### Props

| Name    | Description                | Type               | Default |
| ------- | -------------------------- | ------------------ | ------- |
| color   | customize badge background | `string`           | -       |
| value   | badge text                 | `string \| number` | -       |
| numeric | show more                  | `boolean`          | `false` |
| max     | max of value               | `number`           | `99`    |

### Slots

| Name    | Description       |
| ------- | ----------------- |
| default | customize content |
